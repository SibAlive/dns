1. Данный сайт доступен для просмотра из любой точки мира по адресу: http://sibalive.sytes.net:12001/ 
2. Сайт расположен на моем личном серверном компьютере (ubuntu server) с применением таких утилит как: Docker, nginx, Postgres и др. 
3. При пуше изменений на GitHub, сервер автоматически деплоит изменения с помощью CI/CD пайплайна. Также автоматически применяется миграция с помощью alembic. 
4. Онлайн магазин, полностью написанный мною с помощью библиотеки Flask (Python) (front часть написана с использованием ИИ) 
5. Данный сайт позволяет регистрироваться пользователям, имеет избранное и корзину (содержимое избранного и корзины переносятся при регистрации) 
6. На сайте присутсвуют категории, подкатегории и товары. Также реализован поиск по товарам, имеется построение url истории (breadcrumps) 
7. Сайт имеет админ панель по адресу /admin (Для предоставления логина и пароля потенциальному работодателю, прошу связаться со мной по номеру телефона 8-923-444-30-55, либо по e-mail: Roshchupkinab@mail.ru). 
8. В админ панели реализована возможность управлением категориями, подкатегориями и товарами, пользователями и заказами.

# Установка и запуск приложения
1. Клонирование репозитория:
# git clone https://github.com/SibAlive/dns.git
# cd dns
2. Создание виртуального окружения (для Windows):
# python -m venv venv
# venv/Scripts/activate
3. Установка зависимостей:
# pip install -r requirements.txt
4. Настройка переменных окружения (файл .env необходимо заполнить своими данными:
# cp .env.example .env
5. Восстановления дампа БД
# pg_restore -h localhost -p 5432 -U postgres -d test_dns --clean --if-exists backup_test.dump
6. Отметить папку dns как source root, запустить сервер локально:
# waitress-serve --host=localhost --port=5000 "wsgi:application"
