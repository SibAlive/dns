Данный сайт доступен для просмотра из любой точки мира по адресу: http://sibalive.sytes.net:12001/
Сайт расположен на моем личном серверном компьютере (ubuntu server) с применением таких утилит как: Docker, nginx, Postgres и др.
При пуше изменений на GitHub, сервер автоматически деплоит изменения с помощью CI/CD пайплайна. Также автоматически применяется миграция с помощью alembic
Онлайн магазин, полностью написанный мною с помощью библиотеки Flask (Python) (front часть написана с использованием ИИ)
Данный сайт позволяет регистрироваться пользователям, имеет избранное и корзину (содержимое избранного и корзины переносятся при регистрации)
На сайте присутсвуют категории, подкатегории и товары. Также реализован поиск по товарам, имеется построение url истории (breadcrumps)
Сайт имеет админ панель по адресу /admin (Для предоставления логина и пароля потенциальному работадателю, прошу связатьс со мной по номеру телефона 8-923-444-30-55, либо по e-mail: Roshchupkinab@mail.ru). 
В админ панели реализована возможность управлением категориями, подкатегориями и товарами, пользователями и заказами.
